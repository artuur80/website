For the first project we made a game variant of Hearthstone called HowestStone. This was the first time that we had to use our skills of what we learned in the other courses. Because it was such a big project we learned a lot while doing the project. Some things went wrong. For example some of our members focused on making the game look good, while the game was not yet playable. This lost us precious time. Also we noticed in this project how important it is to ask question where we were stuck. In this project I worked together with three colleagues of mine: Bert Van Mulders, Bram Vermeulen en Brend Lambert.
